# mern-blog-app
MERN Blog API

REST API for a blog application built with Node.js, Express, MongoDB and JWT authentication.

Features
User registration & login
JWT authentication
Create, read, update, delete posts
Upload images (Multer)
View count tracking

Tech Stack
Node.js
Express
MongoDB (Mongoose)
JWT
Multer

Installation
git clone https://github.com/your-username/mern-blog-api.git
cd mern-blog-api
npm install
npm run start:dev

API Endpoints

Auth
POST /auth/register
POST /auth/login
GET /auth/me

Posts
GET /posts
GET /posts/
POST /posts
PATCH /posts/
DELETE /posts/
